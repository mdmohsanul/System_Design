## Step 1: http methods

Master the HTTP methods. You should know the difference between GET, PUT, POST, DELETE, PATCH and OPTIONS. You should know how to use these methods and what each method does.

Try with different tools like fetch, postman, axios etc.

## Step 2: Status code and request headers

You should know what status code are and what they mean. You should know how to handle different status codes in your code. You should know how to pass request headers. You should know what headers are and whatthey do. How to set headers in your code.

- request headers
- request ip
- request user agent
- path variables
- query parameters

## Step 3: Response handling

You think you know how to handle responses. But do you know how to handle different types of responses? You should know how to handle JSON, XML, and other types of responses. You should know how to handle different types of errors. You should know how to handle different types of responses in your code. Learn things like

- response headers
- response caching
- response compression(gzip, brotli, etc) in complex app like zerodha we get data in gzip format means compressed format.
- response cookies
- redirect handling
- handle images (jpeg, png, webp, svg, etc)
